功能一：商品管理

支持商品的增删改查，包括商品名称、价格、描述、图片等信息的管理， 以及商品分类的动态组合查询和分页展示。

🛡️

功能二：用户权限

基于 Spring Security 实现用户认证与授权，支持密码登录和邮箱验证码登录， 不同角色拥有不同的菜单权限。

⚡

功能三：双重缓存

结合 Spring Cache 声明式缓存与 Redis 编程式缓存，有效减少数据库访问压力， 提升系统高并发性能。
